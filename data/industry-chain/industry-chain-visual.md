## 数据格式说明

### 行业配置格式

```yaml
# industry-chain-data.yaml
industries:
  - name: "新能源汽车"
    market: "a"
    keywords:
      - "新能源"
      - "电动汽车"
      - "电动车"
      - "锂电池"
      - "动力电池"
      - "购车补贴"
      - "购置税"
    chain:
      downstream:
        stocks:
          - { code: "002594", name: "比亚迪" }
          - { code: "000625", name: "长安汽车" }
        coefficient: 1.0
      midstream:
        stocks:
          - { code: "300750", name: "宁德时代" }
          - { code: "002074", name: "国轩高科" }
        coefficient: 0.8
      upstream:
        stocks:
          - { code: "002460", name: "赣锋锂业" }
          - { code: "002466", name: "天齐锂业" }
        coefficient: 0.6
```
