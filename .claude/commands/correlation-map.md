搜索网络，识别当前跨资产异常关联性，找出历史信号含义和受益于正常化的交易机会。

搜索数据源：
1. TradingView：tradingview.com（查看多资产价格走势对比）
2. FRED 关联数据：fred.stlouisfed.org（VIX、债券收益率、黄金、DXY）
3. Macrotrends：macrotrends.net（历史关联性数据）
4. 近期宏观评论：Bloomberg、Reuters 关于跨资产异动的报道

需要关注的异常关联（检查是否出现）：
- 黄金 + 股市同时上涨（通常负相关）
- 债券 + 股市同时下跌（通常负相关，即"risk-off"失灵）
- 美元 + 黄金同时上涨（通常负相关）
- 美元 + 新兴市场股市同时上涨（通常负相关）
- 油价 + 美股正相关失效（正常时衰退预期同跌）
- 高收益债 + 国债利差急速收窄（信用风险与安全资产同跑）

分析框架：
1. **当前异常清单**：列出当前观察到的关联性异常（附价格走势截图说明或数据描述）
2. **历史解读**：每个异常在历史上通常意味着什么（流动性危机 / 政策转向 / 通胀预期变化等）
3. **受益于正常化的3个交易**：如果关联性回归历史均值，哪些交易可以获益
   - 交易方向、标的、入场逻辑
4. **数据来源**：每个异常附具体数据来源链接
