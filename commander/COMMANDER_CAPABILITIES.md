# COMMANDER_CAPABILITIES.md - Commander 能力扩展

## 必须具备

### memory - 历史记忆
- **功能**: 记住项目历史进展、决策、风险记录
- **存储**: `memory/` 目录下的结构化文件
- **用途**: 做对比、回顾、趋势分析

### summarize - 汇总压缩
- **功能**: 将各 Agent 的输出压缩成简洁汇报
- **实现**: 使用 summarize skill
- **用途**: 汇报前的信息压缩

### tavily - 金融搜索
- **功能**: 搜索金融行业动态、市场背景
- **API**: Tavily Search API
- **用途**: 提供市场 context，汇报更有深度

## 非常有用

### cron / scheduler - 定时任务
- **功能**: 定时自动汇报（日报/周报）
- **触发**: 每天早上 9:00 自动生成汇报
- **配置**: HEARTBEAT.md 中定义

### notify - 主动推送
- **功能**: 重要风险时主动推送通知
- **渠道**: 微信/邮件/Telegram (可选)
- **触发条件**: 阻塞风险、Bug 激增、重大延误

## 锦上添花

### chart / 可视化
- **功能**: 把进度和数据可视化
- **类型**: 燃尽图、进度条、状态分布

### diff - 状态对比
- **功能**: 对比两个时间点的状态变化
- **用途**: "上周 vs 本周" 清晰可见
