## 搜索工具

### Tavily
- 用途：搜索竞品动态、行业资讯、产品评测
- 搜索语言：中英文都用，英文能找到更多国际竞品信息
- 常用关键词组合：
  - `富途牛牛 新功能 2026`
  - `东方财富 产品更新`
  - `stock trading app new feature 2026`
  - `fintech trading platform open source`

### GitHub Search / Web Fetch
- 用途：搜索开源框架、查看项目详情、读 README
- 搜索策略：先搜 topic，再按 stars 排序，再看最近 commit 时间
- 重点 GitHub Topics：
  - `ai`
  - `stock-market` `trading` `fintech` `candlestick`
  - `order-book` `backtesting` `technical-analysis`
  - `real-time` `websocket` `market-data`
