# 股票行情受新闻影响的研究报告

## 1. 研究背景与意义

新闻是影响股票短期走势的重要因素之一。研究表明，新闻事件能够通过多种机制影响股价：
- 改变市场对公司盈利的预期
- 引发情绪波动和资金流动
- 导致基本面的重新评估

## 2. 新闻影响股价的核心机制

| 影响类型 | 机制描述 | 影响周期 |
|---------|---------|---------|
| **预期改变** | 新闻改变市场对公司未来的盈利预期 | 中短期 |
| **情绪传导** | 恐慌/乐观情绪在市场蔓延 | 短期 |
| **资金流动** | 机构调仓、散户跟随 | 短期-中期 |
| **基本面重估** | 政策、行业变化影响估值模型 | 中长期 |

## 3. 关键研究维度

### 3.1 情绪因子构建
- 新闻正负面情感评分
- 关键词热度（政策、业绩、并购等）
- 社交媒体情绪指数

### 3.2 事件驱动策略
- 公告发布后的异常收益 (CAR)
- 成交量异动检测
- 时间序列回归

### 3.3 金融领域专用词典
根据 Loughran & McDonald 的研究，金融文本中某些词汇的情感含义与普通英语不同：
- **负面词**：liability, loss, decline, bankruptcy（这些词在金融语境中才有负面含义）
- **正面词**：growth, increase, profit, innovation

## 4. 数据来源建议

| 数据源 | 用途 |
|--------|------|
| 财新、同花顺 | 机构新闻 |
| 微博、雪球 | 社交情绪 |
| 东方财富 | 股吧舆情 |
| 政策文件 | 宏观影响 |
| Reuters | 国际化新闻 |

## 5. 技术实现方案

### 5.1 整体架构
```
新闻采集 → 预处理 → 情感分析 → 因子构建 → 策略回测 → 信号生成
```

### 5.2 关键技术点
1. **文本向量化**: Word2Vec, GloVe, BERT
2. **分类模型**: CNN, LSTM, Bayesian NN
3. **事件检测**: 关键词匹配 + 异常标注
