# 市场数据收集规则 - 完整说明

## 一、当前收集的数据类型

### 1. K线数据
- **来源**: 
  - A股: Twelvedata API
  - 港股: 腾讯财经API (gtimg.cn)
  - 美股: Twelvedata API
- **时间框**: 5m, 15m, 1h, 4h, 1d, 1w, 1M
- **范围**: 3个月历史数据

### 2. 实时行情
| 市场 | 数据项 | 频率 |
|------|--------|------|
| 港股 | 价格/涨跌幅/成交量/买卖价 | 实时 |
| 美股 | 价格/涨跌幅/成交量 | 实时 |
| A股 | 价格/涨跌幅 | 实时 |

### 3. 技术指标 (自建库)
- Bollinger Bands (-3到+3评级)
- RSI (14周期)
- MACD
- 趋势判断 (上涨/下跌/震荡)
- K线形态识别

### 4. 资金流向
- 现有: 收盘后资金流入分析
- 来源: 各市场API

## 三、触发式收集

### 立即推送条件
- **短线信号≥70分** → 飞书立即推送
- **放量突破新高** (+30分)
- **均线多头排列** (+15分)
- **RSI超卖反弹** (+15分)

### 飞书技术查询
- 用户发送 "指标 AAPL" → 实时查询技术指标
