## 二、核心竞品详情

### 1. tradingview-mcp (⭐558, Fork 144)
**定位**: 高级 TradingView MCP Server，AI 驱动的市场分析框架

**核心功能**:
| 类别 | 功能 |
|------|------|
| 📊 回测引擎 | 6种策略 (RSI/Bollinger/MACD/EMA Cross/Supertrend/Donchian)，含 Sharpe/Calmar/Expectancy 指标 |
| 💰 实时行情 | Yahoo Finance 实时报价 (股票/加密货币/ETF/外汇) |
| 🧠 情绪分析 | Reddit 社区情绪 + RSS 新闻 (Reuters/CoinDesk) |
| 📈 技术分析 | 30+ 技术指标 (RSI/MACD/Bollinger Bands) + 15种 K 线形态识别 |
| 🌍 多交易所 | Binance/KuCoin/Bybit+/NASDAQ/NYSE/EGX(埃及)/BIST(土耳其) |

**技术栈**: Python 3.10+ / MCP (Model Context Protocol) / Claude Desktop / OpenClaw
**部署**: pip install / Docker / OpenClaw 集成 (Telegram/WhatsApp/Discord)

**特色**: 
- 多代理系统：Technical Analyst + Sentiment Analyst + Risk Manager 实时辩论
- 输出 BUY/SELL/HOLD + 置信度评分
- 支持 Walk-forward backtesting (规划中)

### 2. InvestAI (⭐153, Fork 34)
**定位**: 面向散户的 AI 投资助手，专注规则执行与信号提醒

**核心理念**: 
> "Let AI execute the rules. Keep decisions in human hands."

**核心功能**:
- ✅ 基于固定规则系统分析持仓股 (趋势/回调/风险条件)
- ✅ 定时监控 + 关键时点通知 (Slack/Lark/WeWork)
- ✅ 交互式实时分析 (通过 Cherry Studio + MCP)
- ❌ 不提供自动交易、不做价格预测、不给"必涨"建议

**策略示例**:
- 市场环境评估 (整体趋势判断)
- 个股趋势突破条件 (MA20/MA60, 20日阻力位)
- 成交量确认 (20日均量)
- RSI 动量过滤 (40-65区间)
- CCI 趋势健康检查

**部署**: Docker 一键启动

## 三、新兴特性与趋势

### 1. MCP (Model Context Protocol) 崛起
- 成为 AI 助手与外部工具交互的标准协议
- tradingview-mcp 已深度集成 Claude Desktop / OpenClaw

### 2. 多代理协作架构
- Technical Analyst / Sentiment Analyst / Risk Manager 独立 agents 辩论
- 输出结构化决策 (BUY/SELL/HOLD + confidence score)

### 3. 回测平民化
- 过去需要专业工具的策略回测，现在开源项目即可完成
- 支持 Sharpe Ratio / Calmar Ratio / Max Drawdown 等机构级指标

### 4. 情绪分析集成
- Reddit 社区情绪量化
- RSS 新闻实时抓取
- 多源数据融合 (技术面 + 基本面 + 情绪面)

### 5. 多交易所支持
- 加密货币: Binance/KuCoin/Bybit
- 股票: NASDAQ/NYSE
- 新兴市场: 土耳其 BIST / 埃及 EGX

## 四、对 MarketPlayer 的启示

1. **技术分析先行**: tradingview-mcp 的 30+ 指标库是基础能力，MarketPlayer 应快速补齐
2. **MCP 集成**: 考虑支持 MCP 协议以对接 Claude/OpenClaw 等 AI 助手
3. **情绪分析差异化**: Reddit + 新闻情绪是竞品重点方向，可考虑中文社区 (雪球/东方财富)
4. **回测能力**: 6 种策略回测 + 多策略对比是硬核功能，建议作为 Phase 2 目标
5. **不做交易执行**: 竞品普遍强调"只提供信号，不执行交易"，这与 MarketPlayer 定位一致
