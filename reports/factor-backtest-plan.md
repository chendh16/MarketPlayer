# 产业链因子回测验证方案

## 1. 验证目标

验证产业链因子对股价的预测能力，确认因子有效性。

---

## 2. 验证指标

### 2.1 因子有效性指标

| 指标 | 公式 | 合格标准 |
|------|------|---------|
| **IC (Information Coefficient)** | IC = corr(因子值, 下期收益) | IC > 0.02 |
| **IR (Information Ratio)** | IR = mean(IC) / std(IC) | IR > 0.5 |
| **胜率** | 因子_top20% 收益 > 因子_bottom20% 收益 | 胜率 > 55% |
| **分组回测** | 按因子分5组，看收益单调性 | 线性递增/递减 |

### 2.2 传导验证指标

| 指标 | 验证方式 | 合格标准 |
|------|---------|---------|
| **下游领先** | 下游股票涨幅 vs 上游股票涨幅 | 下游反应更快 |
| **系数准确性** | 实际收益 / 理论系数 | 误差 < 30% |
| **行业联动** | 同行业股票相关性 | 相关系数 > 0.6 |

## 4. 测试场景

### 场景1: 行业利好新闻测试

**新闻**: "国务院出台新能源汽车购置税减免政策"

**预期**:
1. 下游整车股（比亚迪）涨幅 > 中游（宁德时代）> 上游（赣锋锂业）
2. 反应时间：下游 1-3天，中游 3-5天，上游 5-7天

**验证方法**:
```
1. 标记新闻发布日期 T
2. 计算 T+1, T+3, T+5, T+10 各股票涨幅
3. 验证涨幅顺序: 下游 > 中游 > 上游
```

---

### 场景2: 行业利空新闻测试

**新闻**: "某新能源车企销量大幅下滑"

**预期**:
1. 下游跌幅最大，向上游传导
2. 传导系数生效：下游 -5% → 中游 -4% → 上游 -3%

---

### 场景3: 轮动测试

**场景**: 跨行业比较

| 行业 | 新闻事件 | 预期因子得分 | 验证方向 |
|------|---------|------------|---------|
| 新能源汽车 | 政策利好 | +0.5 | 因子正向 |
| 房地产 | 调控收紧 | -0.4 | 因子负向 |
| 医药 | 集采降价 | -0.3 | 因子负向 |
| 食品饮料 | 白酒提价 | +0.3 | 因子正向 |

## 5. 因子计算公式

### 5.1 产业链因子

```
因子得分 = 新闻情感 × 传导系数 × 时间衰减

其中:
- 新闻情感: -1 ~ +1
- 传导系数: 下游=1.0, 中游=0.8, 上游=0.6
- 时间衰减: 半衰期 3-7天
```

### 5.2 累积因子

```
累积因子 = Σ(因子得分 × 权重) / Σ权重

时间窗口: 5日
权重: T-0=0.4, T-1=0.3, T-2=0.15, T-3=0.1, T-4=0.05
```

## 7. 数据需求

### 7.1 必要数据

| 数据 | 来源 | 频率 | 量级 |
|------|------|------|------|
| 日K线 | 现有 | 日 | 100+股票 |
| 新闻数据 | 东方财富/同花顺 | 日 | 1000+条/日 |
| 产业链映射 | 现有模块 | - | 75只股票 |

### 7.2 数据格式

```typescript
// 因子数据
interface FactorData {
  date: string;           // 2024-01-15
  symbol: string;         // 600519
  industry: string;        // 食品饮料
  chainLevel: 'downstream' | 'midstream' | 'upstream';
  sentimentFactor: number; // -1 ~ 1
  heatFactor: number;      // 0 ~ 1
  chainFactor: number;    // -1 ~ 1 (产业链)
  compositeScore: number;  // 综合得分
}

// 收益数据
interface ReturnData {
  date: string;
  symbol: string;
  forwardReturn: number;  // 下期收益
}
```

## 10. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 数据不足 | 无法验证 | 增加股票池 |
| IC偏低 | 因子无效 | 调整系数/权重 |
| 传导方向反 | 逻辑错误 | 复核产业链图 |
| 过拟合 | 实盘失效 | 用不同周期验证 |
