## 通道一：事件驱动 (Event-Driven)

### 事件类型与响应

| 事件类型 | 关键特征 | 时间窗口 | 信号强度 |
|---------|---------|---------|---------|
| **业绩预告/公告** | 盈利大幅上升/下降 | 1-3天 | ⭐⭐⭐⭐⭐ |
| **政策发布** | 行业利好/利空 | 1-4周 | ⭐⭐⭐⭐ |
| **并购重组** | 重大资产变动 | 1-2周 | ⭐⭐⭐⭐ |
| **预警/退市** | 风险提示 | 即时-3天 | ⭐⭐⭐⭐⭐ |
| **高管变动** | 董事长/核心人物 | 1-5天 | ⭐⭐⭐ |
| **诉讼/处罚** | 监管处罚 | 1周 | ⭐⭐⭐⭐ |
| **产品突破** | 新药/新技术 | 1-4周 | ⭐⭐⭐ |

## 通道二：因子化 (Factor-Based)

### 因子体系

```
情感因子 ──┬──> 新闻情感得分 (日/周)
           ├──> 情感变化率 (δ情感)
           └──> 情感偏离度 (相对行业)

热度因子 ──┬---> 新闻发布数量 (日频)
           ├──> 关键词命中次数
           └──> 媒体关注度 (媒体层级加权)

关注度因子 ┬──> 社交讨论量 (雪球/微博)
           ├──> 搜索指数 (百度)
           └──> 机构研报数量
```

### 时间衰减函数

```typescript
// 指数衰减模型
function newsDecay(age: number, halfLife: number): number {
  return Math.pow(0.5, age / halfLife);
}

// 不同事件类型的半衰期
const halfLifeConfig = {
  '业绩公告': 3,      // 3天衰减一半
  '政策发布': 7,      // 7天
  '日常新闻': 1,      // 1天
  '突发事件': 0.5,    // 12小时
  '研报观点': 14      // 14天
};

// 综合因子计算
function factorWithDecay(newsList: News[], halfLife: number): number {
  let score = 0;
  for (const news of newsList) {
    const decay = newsDecay(news.age, halfLife);
    score += news.sentiment * news.weight * decay;
  }
  return score;
}
```

## 信号融合策略

### 融合公式

```
FinalSignal = α × EventSignal + (1-α) × FactorSignal

其中:
- α = 基于事件置信度动态调整
- EventSignal = 事件驱动通道输出 [-1, 1]
- FactorSignal = 因子化通道输出 [-1, 1]
```

### 权重学习

```python
# 简单加权融合（可后续优化为ML模型）
def fusion(event_signal, factor_signal, event_confidence):
    alpha = event_confidence  # 事件置信度越高，事件通道权重越大
    
    final = alpha * event_signal + (1 - alpha) * factor_signal
    
    # 阈值过滤
    if abs(final) < 0.2:
        return 'hold'
    elif final > 0.2:
        return 'buy'
    else:
        return 'sell'
```
