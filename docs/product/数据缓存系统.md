# 数据缓存系统 - 已完成

## 功能

| 功能 | 说明 |
|------|------|
| 本地缓存 | `src/services/data/cache.ts` |
| 智能读取 | 优先使用缓存，过期自动更新 |
| 批量更新 | 批量获取并缓存历史数据 |
| 过期清理 | 自动清理7天前缓存 |
| CSV导出 | 支持导出为CSV格式 |

## 缓存策略

- 首次请求: 自动获取并保存到本地
- 后续请求: 优先读取本地缓存
- 24小时后: 自动更新过期数据
- 7天后: 自动清理过期文件
