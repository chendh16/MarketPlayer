## 二、新兴特性与趋势

### 1. 多模态文档理解
- **WallStr.Chat**: 支持并行阅读数十个PDF，适用投行/PE场景
- **FinGenius**: 集成Jupyter进行财务分析和自动化决策

### 2. 新闻情绪驱动
- **Sentimeter**: 印尼股票交易所自动化，分析新闻情绪生成每日信号
- **Stocex**: 市场新闻→股价预测，适用于日内交易

### 3. 多智能体协作
- **Stock-Analysis-Bot**: 多智能体分析美国和印度市场信号
- **velloregrao/agents**: AI股票交易+Teams机器人集成

### 4. 日报/简报系统
- **alpha-brief**: 每日股票简报+TradingAgents

### 5. 实时WebSocket API
- **aitrados-api**: 多时间周期OHLC、经济事件WebSocket API
