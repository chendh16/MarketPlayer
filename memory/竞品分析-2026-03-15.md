## 一、核心竞品汇总

### 1. Stock-Market-AI-GUI (347 ⭐)
**定位**: 股票预测与量化交易机器人  
**技术栈**: Python, Django, TensorFlow, Keras, LSTM, 强化学习

**核心功能**:
- LSTM 模型进行股价预测
- 强化学习交易 Agent
- Web 界面可视化
- 支持股票代码查询和数据可视化

### 2. WeFinance (130 ⭐)
**定位**: AI 个人财务助手  
**技术栈**: Streamlit, GPT-4o Vision, LangChain

**核心功能**:
- 📸 账单图像识别 (GPT-4o Vision, 100%准确率)
- 💬 对话式财务顾问
- 📊 可解释 AI 投资建议
- 🚨 异常消费检测

**技术亮点**:
- 用 Vision LLM 替代传统 OCR (PaddleOCR 0% → 100%)
- XAI (可解释AI) 作为核心架构而非附加功能
- 混合规则引擎 + LLM 提供透明决策链

## 三、对 MarketPlayer 的启示

1. **多模态输入**: 票据/截图识别是个人理财的痛点，Vision LLM 是可行方案
2. **可解释性很重要**: 用户需要理解 AI 推荐理由，混合规则引擎值得借鉴
3. **Agent 架构**: 考虑数据收集 → 分析 → 建议的多 Agent 协作
4. **自动化触发**: Telegram/飞书机器人集成适合实时提醒
5. **RAG 应用**: SEC/财报智能分析是机构用户的强需求
